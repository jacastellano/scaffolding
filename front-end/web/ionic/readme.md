
# Prerequisites

* Node.js
    - install LTS version
    - v10.15.1

* npm
    - 6.7.0

* ionic
    ```
    $ npm install -g ionic
    ```
    - ionic@4.10.2


# Steps to build it

1. Install the npm packages described in the ./package.json

    ```
    $ npm install
    ```

2. Running
    ```
    $ ionic serve
    ```
